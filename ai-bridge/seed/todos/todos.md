# Todos

Quick personal reminders for whoever drives this control panel. Capture with
`/todo <text>`, close one with `/todo done <text>`, list with `/todo`. These are
**lightweight notes** — for formal, agent-executed work use `/new-project` and the
task lifecycle under `projects/` instead (promote a todo there once it's real work).
